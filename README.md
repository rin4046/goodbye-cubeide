# Goodbye CubeIDE

An extension for people who don't like CubeIDE.

## Requirements

- STM32CubeIDE
- GNU Make

## Settings

- `goodbye-cubeide.cubeIdePath`
- `goodbye-cubeide.cubeIdeWorkspacePath`
